下载<a href="https://github.com/google-research/albert">albert</a>到项目路径<br />
创建albert_model路径，将预训练好的albert模型放到这个路径下解压<br />
具体结构如下：
<img src="imgs/struct.png">
python3 run.py --mode xxx <br />
